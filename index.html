TRANSLATOR

Previous Next

The Translator app, a Python-based project in my portfolio, incorporates a backend written in Python. Leveraging the Google Translate API, it efficiently handles translation requests, including the translation of comments within source files. Seamlessly connecting users to the powerful translation capabilities of Google Translate, this app enables accurate and convenient translations. The frontend, built using Zenity in Bash, provides a user-friendly interface for effortless translation of text, including comments in source files.
